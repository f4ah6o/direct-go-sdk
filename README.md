# direct-go-sdk

[![CI](https://github.com/f4ah6o/direct-go-sdk/actions/workflows/ci.yaml/badge.svg)](https://github.com/f4ah6o/direct-go-sdk/actions/workflows/ci.yaml)

Go SDK for Direct4B (direct4b.com) chat platform, providing both a low-level SDK (`direct-go`) and a high-level bot framework (`daab-go`).

## Modules

- **[direct-go](./direct-go)**: Direct4B Go SDK - WebSocket/MessagePack RPC client
- **[daab-go](./daab-go)**: Bot framework and CLI tool built on direct-go
